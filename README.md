# Customer Sales Analysis – Power BI Dashboard

This repository contains an interactive **Customer Sales Analytics** dashboard built using **Power BI**.

##  Features
- Customer-wise sales insights  
- KPIs & slicers  
- Trend analysis  
- Clean UI & visual storytelling  

##  Tech Used
- Power BI Desktop  
- DAX Measures  
- Data Modeling  

##  Project Structure
```
/data        → raw or sample datasets
/images      → dashboard screenshots
/docs        → additional notes
customer_sales_k.pbix → main Power BI report
README.md    → project documentation
```
## What to Expect in This Project
This project provides a complete end-to-end analysis of Customer Sales Performance using Power BI. Inside the repository, you will find everything needed to understand the business problem, explore the data, and review the interactive dashboard.

1. Clean & Structured Project Files

Power BI report (customer_sales_k.pbix)

Dashboard preview image

Business documents explaining the problem

SQL scripts and supporting files (if used)

2. Fully Designed Power BI Dashboard

A professionally built interactive dashboard containing:

Customer-wise revenue insights

Category-level performance

Sales KPIs and metrics

Discount analysis

Shipping and order behavior

Product rating patterns

3. Data Modeling & DAX

You will see:

Star-schema based data model

Calculated columns

Custom DAX measures for KPIs, trend analysis, and segmentation

4. Business Insights

The dashboard highlights key findings such as:

Revenue contribution by customer segments

Discount impact on sales

Performance of product categories

Shipping preference analysis

Loyal vs New customer patterns

5. How It Helps

This project is useful for anyone looking to learn:

Power BI dashboard building

Data modeling techniques

Business analytics storytelling

Creating end-to-end BI solutions
